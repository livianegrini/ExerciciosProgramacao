10) Crie um exemplo prático no qual você tenha duas classes:

- Uma classe `Produto` com atributos `nome` e `preco`, e um método `calcularDesconto()` que aplica um desconto fixo de 10% no preço do produto.
- Uma classe `Livro` que herda de `Produto` e modifica o método `calcularDesconto()`, aplicando um desconto de 20% no preço dos livros.

Explique como funciona a herança nesse contexto e como você implementaria a modificação do método na classe `Livro`.

```javascript
class Produto {
  constructor(nome, preco) {
    this.nome = nome;
    this.preco = preco;
  }

  calcularDesconto() {
    // Aplicando 10% de desconto
    return this.preco * 0.9; 
  }
}

class Livro extends Produto {
  constructor(nome, preco) {
    super(nome, preco);
  }

// Aplicando 20% de desconto
  calcularDesconto() {
    return this.preco * 0.8; 
  }
}

// Testando a implementação
const produto = new Produto("Celular", 800);
console.log(`Preço com desconto do produto: R$ ${produto.calcularDesconto()}`);

const livro = new Livro("JavaScript Avançado", 200);
console.log(`Preço com desconto do livro: R$ ${livro.calcularDesconto()}`);

```

A herança permite que Livro reutilize atributos e métodos de Produto, mas sobrescrevendo calcularDesconto() para aplicar um desconto de 20%.
